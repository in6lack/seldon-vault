## Что такое Seldon Vault?

Seldon Vault — это бесплатный публичный движок геополитических прогнозов на базе искусственного интеллекта. Он не опирается на ставки людей, как рынки предсказаний (Polymarket, Metaculus), и не даёт одноразовые ответы, как ChatGPT. Вместо этого используется **мультиагентная ИИ-система**: 11 специализированных аналитиков — включая противоположные пары Ястреб/Голубь и Бык/Медведь — независимо исследуют мировые события с разных перспектив, адверсарный Скептик проверяет факты, а Арбитр Селдон синтезирует сильнейшие прогнозы через итеративное ReACT-рассуждение с доступом к 6 инструментам в реальном времени.

Каждый прогноз содержит полные обоснования от каждого агента, вероятность от 5% до 95% и временной горизонт. Вероятности обновляются каждые 6 часов по формуле Байеса при поступлении новых данных. Точность отслеживается публично через [Brier Score](docs/accuracy.md) — потому что предсказание без трек-рекорда — это просто мнение.

Помимо ежедневных прогнозов, ежемесячный пайплайн **Seldon Plan** создаёт структурные прогнозы на горизонте 1-10 лет — анализируя волны Кондратьева, гегемонические переходы и демографические сдвиги через 6 аналитиков-футурологов.

Название проекта отсылает к [Хари Селдону](https://ru.wikipedia.org/wiki/%D0%A5%D0%B0%D1%80%D0%B8_%D0%A1%D0%B5%D0%BB%D0%B4%D0%BE%D0%BD) — математику из серии романов Айзека Азимова «Основание», который создал **психоисторию** — математический фреймворк для предсказания будущего поведения больших популяций. Настоящая психоистория пока остаётся научной фантастикой, но Seldon Vault исследует, что возможно, когда несколько ИИ-агентов с разной экспертизой и противоположными когнитивными установками совместно анализируют одни и те же мировые события.

## Как это работает

Seldon Vault запускает два пайплайна: **ежедневный** для краткосрочного прогнозирования и **ежемесячный структурный** (Seldon Plan) для сценариев на десятилетия.

### Ежедневный пайплайн

1. **Непрерывный сбор сигналов** — 4 раза в день из 12 источников: RSS-ленты (~63 фида: Reuters, BBC, Al Jazeera, SCMP), GDELT, ACLED, FRED, Metaculus & Polymarket, GDACS, Fear & Greed Index, Telegram-каналы, Reddit и Bluesky. Инкрементальный фетчинг с Redis-дедупликацией предотвращает избыточный сбор
2. **Обработка сигналов** — ИИ-классификатор категоризирует каждый сигнал по сектору, тональности, важности (0-100) и временному масштабу. Сигналы с важностью ниже 30 отфильтровываются
3. **Граф знаний** — Три уровня семантической структуры: кластеризация сигналов (группировка схожих сигналов с подсчётом независимых источников), рейтинги надёжности источников (точность на базе Brier Score), цепочки событий (кросс-дневная привязка с 8 стадиями жизненного цикла от слуха до последствий, плюс матрица плотности с конкурирующими интерпретациями)
4. **Мультиагентный анализ** — 11 аналитиков параллельно исследуют сигналы. Три ключевых домена (геополитика, экономика, политика) используют **двойные персоны** — Ястреб и Голубь с противоположными когнитивными установками. Пять сольных доменов могут использовать мультимодельный LLM-совет из 3 провайдеров. Каждый аналитик получает поведенческие профили лидеров, обратную связь калибровки, память похожих прогнозов и контекст цепочек событий
5. **Слой слияния** — Пары Ястреб/Голубь сопоставляются по заголовкам и объединяются в обогащённые предложения с метриками расхождения и квантовой интерференцией персон — без единого вызова LLM
6. **Проверка Скептика** — Двухуровневый адверсарный обзор: быстрый Скептик (префильтр с 6 структурными kill-правилами) + основной Скептик с поиском (Tavily) для фактчекинга. Предложения с оценкой ниже 50/100 автоматически отклоняются
7. **Синтез Селдона** — Арбитр использует **ReACT-рассуждение** (итеративные циклы Мысль-Действие-Наблюдение) с 6 инструментами: поиск исторических аналогий, запрос экономических индикаторов, веб-фактчекинг, исследование цепочек событий, проверка трек-рекорда агентов, сравнение предложений. Выбирает топ 3-7 прогнозов, калибрует вероятности, обнаруживает Кризисы Селдона и Каскадные Нарративы
8. **Эвристические алерты** — 6 детерминированных проверок на сверхуверенность, якорение в среднем диапазоне, разногласия аналитиков, красные флаги Скептика, временные несоответствия и устаревшие предсказания
9. **Перевод** — Все агенты выдают результат только на английском. Выделенный слой перевода конвертирует все пользовательские поля в русский через один вызов LLM на прогноз для когерентности контекста
10. **Каскадная пропагация** — Обнаруживаются причинно-следственные цепочки между прогнозами, вероятности распространяются через граф нарративов. Квантовое моделирование интерференции вычисляет конструктивные/деструктивные эффекты
11. **Байесовские обновления** — Каждые 6 часов вероятности обновляются через Likelihood Ratios. Макс. сдвиг: ±15% в день
12. **Отслеживание точности** — Brier Score для каждого разрешённого прогноза. Точность агентов возвращается в калибровку и веса
13. **Авторезолюция** — Проверка по реальным данным (FRED, Yahoo Finance, курсы валют, World Bank) и новостным источникам

### Ежемесячный структурный пайплайн (Seldon Plan)

1-го числа каждого месяца запускается отдельный пайплайн для долгосрочных (1-10 лет) структурных прогнозов:

1. **Сбор данных** — World Bank API, МВФ WEO, население ООН, OWID/GISS для 15 приоритетных стран
2. **Синтез мирового состояния** — LLM сжимает 90 дней истории прогнозов в доменные брифы
3. **Структурный RAG** — Исторические аналогии через pgvector (волны Кондратьева, гегемонические переходы, долговые кризисы)
4. **6 аналитиков-футурологов** — Экономика, Геополитика, Технологии, Общество, Климат, Военное дело
5. **Структурный Скептик** — Проверяет каждый домен на 6 «смертельных ловушек» долгосрочного прогнозирования
6. **Структурный Селдон** — Claude Opus синтезирует мастер-сценарии, критические развилки, ведущие индикаторы. Кросс-доменные причинные связи с квантовой интерференцией
7. **Результат** — Отчёт Селдона с мастер-сценариями, критическими развилками, доменными прогнозами и кросс-доменным графом

## ИИ-агенты

### 11 аналитиков (3 двойные пары + 5 сольных)

| Агент | Что анализирует | Режим персоны |
|-------|-----------------|---------------|
| **Геополитик-Ястреб** | Войны, эскалация, жёсткая сила, санкции | Агрессивная оценка рисков |
| **Геополитик-Голубь** | Дипломатия, сотрудничество, деэскалация | Примирительная перспектива |
| **Экономист-Бык** | Рост, устойчивость, рыночные возможности | Оптимистичный взгляд |
| **Экономист-Медведь** | Риски, хвостовые события, долговые кризисы | Пессимистичный взгляд |
| **Политик-Ястреб** | Консолидация власти, репрессии, реализм | Фокус на жёсткой силе |
| **Политик-Голубь** | Институты, реформы, демократическая устойчивость | Фокус на мягкой силе |
| **Технолог** | ИИ, полупроводники, биотех, энергопереход | Мультимодельный LLM-совет |
| **Социолог** | Демография, миграция, социальные движения | Сольная персона |
| **Климатолог** | Климатические риски, экстремальная погода | Сольная персона |
| **Военный аналитик** | Баланс сил, сдерживание, ядерная доктрина | Сольная персона |
| **Аналитик кибербезопасности** | APT-группы, zero-day, вымогатели | Сольная персона |

### Ключевые роли

| Агент | Роль |
|-------|------|
| **Быстрый Скептик** | Быстрый префильтр со структурными kill-правилами (без веб-поиска) |
| **Основной Скептик** | Глубокий адверсарный обзор с Tavily и правом вето |
| **Арбитр Селдон** | Финальный синтез через ReACT-рассуждение с 6 инструментами |

Система двойных персон создаёт подлинное разногласие. Когда Ястреб говорит 80% вероятность конфликта, а Голубь — 35%, сам разброс становится ценной информацией для Арбитра. Персоны объединяются до проверки Скептика — Слой слияния сопоставляет пары Ястреб/Голубь по теме, создаёт обогащённые предложения с обеими перспективами и вычисляет квантовую интерференцию между противоположными точками зрения.

## Пять Столпов Анализа

| Столп | Что делает | Пример |
|-------|-----------|--------|
| **Теория игр** | Моделирует рациональных акторов, матрицы выигрышей, равновесия Нэша | «Рациональный ход страны X — усилить санкции» |
| **Байесовский вывод** | Обновляет убеждения с новыми данными, prior → posterior | «Новые спутниковые данные сдвигают вероятность конфликта на +12%» |
| **Теория хаоса** | Картирует аттракторы, чувствительность, горизонты Ляпунова | «Система приближается к фазовому переходу» |
| **Психоистория** | Статмеханика обществ — циклы, сопоставление паттернов | «Похоже на Азиатский финансовый кризис 1997 года» |
| **Теория сетей** | Каскадные эффекты, графы влияния, пороги перколяции | «Каскадный потенциал: 7/10 — три связанных региона» |

## Бесплатный публичный API

Без аутентификации. Без API-ключа. Без регистрации. Только данные.

```bash
# Сегодняшние прогнозы
curl https://seldonvault.io/api/v1/forecasts/daily

# Все прогнозы с фильтрами
curl "https://seldonvault.io/api/v1/forecasts?sector=geopolitics&status=active"

# Последний Seldon Plan (структурный отчёт)
curl https://seldonvault.io/api/v1/structural/reports/latest

# Обновления в реальном времени через SSE
curl https://seldonvault.io/api/v1/events/stream
```

**30+ эндпоинтов**: прогнозы, метрики, регионы, нарративы, сигналы, цепочки событий, рейтинги источников, аудит пайплайна, структурные отчёты, события в реальном времени. Лимит: 60 запросов/минуту на IP.

> **[Полный гайд по API →](docs/api-guide.md)** | **[Документация API на сайте →](https://seldonvault.io/developers)**

## Ключевые возможности

- **Ежедневные ИИ-прогнозы** из 12 источников данных по 8 секторам
- **Мультиагентный ансамбль** — 11 аналитиков с двойными персонами + Скептик + Арбитр
- **Система двойных персон** — пары Ястреб/Голубь и Бык/Медведь для когнитивного разнообразия с квантовым моделированием интерференции при слиянии
- **ReACT-рассуждение** — итеративное мышление Арбитра Селдона с 6 инструментами в реальном времени
- **Ежемесячный Seldon Plan** — структурные прогнозы на 1-10 лет с мастер-сценариями, критическими развилками и кросс-доменным графом
- **Непрерывный сбор сигналов** — инкрементальный сбор 4р/день с 6-вентильной кластеризацией и ночной сверкой
- **Байесовские обновления** каждые 6 часов
- **Brier Score** — публичный, по агентам, по секторам
- **Самокорректирующаяся система** — калибровка промптов по точности
- **Динамическое взвешивание агентов** — веса надёжности из Brier Score
- **Профилирование лидеров** — 14 поведенческих профилей (Трамп, Путин, Си, Эрдоган, Нетаньяху, Маск и др.)
- **Двуязычность** — EN/RU через выделенный слой перевода
- **Интерактивная карта мира** с региональным анализом рисков
- **Сопоставление исторических аналогий** через RAG
- **7 горизонтов прогнозирования** — от дней до столетий
- **Каскадные Нарративы** — причинно-следственные цепочки с квантовой интерференцией
- **Матрица плотности** — конкурирующие интерпретации цепочек событий с отслеживанием мета-неопределённости
- **Обнаружение Кризисов Селдона**
- **Журнал решений Pipeline**
- **Эвристические проверки** — 6 правил
- **Детекция медиа-предвзятости**
- **Движок авторезолюции** — FRED, Yahoo Finance + ИИ-анализ новостей
- **Граф знаний** — кластеризация сигналов, рейтинги источников, цепочки событий
- **Бесплатный REST API** — 30+ эндпоинтов, без аутентификации
- **SSE** обновления в реальном времени
- **Полностью бесплатный**
